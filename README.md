# 🌾 Farmer Management System

## 📌 Project Overview

The Farmer Management System is a web-based application designed to connect farmers and consumers through a digital marketplace. The system helps farmers manage and sell their products directly to customers, reducing the dependency on middlemen and ensuring better market accessibility.

## 🚀 Features

* 👨‍🌾 Farmer Registration and Login
* 🛒 Consumer Registration and Login
* 📦 Product Management (Add, Update, Delete Products)
* 🔍 Browse and Search Products
* 🛍️ Purchase Products Directly from Farmers
* 👤 User Profile Management
* 📊 Dashboard for Farmers and Consumers
* 📱 Responsive User Interface

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/divya5055/Farmer_Management_System.git
```

### Navigate to the Project Folder

```bash
cd Farmer_Management_System
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

**Windows**

```bash
venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```
